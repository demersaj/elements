# Changelog

## [Version 0.1.0] - 2025-01-XX

### Added

- Initial release of Conditional element
- Support for flexible condition evaluation using JSON configuration
- Multiple comparison operators (equals, greater_than, contains, etc.)
- Nested field access using dot notation
- Dual output routing (true/false) based on condition evaluation
- Support for various data types (strings, numbers, arrays, objects)
