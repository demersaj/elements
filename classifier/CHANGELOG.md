# Changelog

All notable changes to the Classifier element will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of LLM Text Classifier element
- Support for user-defined categories
- Multiple LLM provider options (API, OpenAI, Anthropic)
- Confidence scoring and thresholding
- Custom system prompt support
- Temperature control for classification determinism
- Integration with Frame-based data flow
- Support for text extraction from message and API formats
